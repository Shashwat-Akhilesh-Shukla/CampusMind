# **App Name**: CampusMind

## Core Features:

- Anonymous Login: Allow users to access the app in anonymous mode, bypassing the need for email or phone signup.
- Screening Test: Implement PHQ-9 and GAD-7 short surveys for students to self-assess their mental wellbeing, and return results immediately
- AI Guardian Chatbot: Offer an AI chatbot that can provide support, coping strategies, and mental health tips to students. The chatbot tool decides on useful tips and conversational techniques.
- Counselor Booking: Enable students to anonymously book sessions with counselors through a calendar interface.
- Peer Support Groups: Create a platform for students to connect with each other in moderated support groups, facilitating discussions and shared experiences.
- Language Selector: Implement a language selector with support for English, Hindi, Marathi and Malayalam.

## Style Guidelines:

- Primary color: Soft lavender (#E6E6FA) to promote a calming and supportive environment.
- Background color: Light gray (#F5F5F5) for a clean and modern look.
- Accent color: Light blue (#ADD8E6) to highlight interactive elements and key actions.
- Body and headline font: 'PT Sans', a humanist sans-serif for a modern, clean look and comfortable reading.
- Use simple, clear icons to represent different sections of the app.
- Employ a clean, card-based layout to create a sense of organization, while prioritizing accessibility for users with limited literacy or tech experience.
- Incorporate subtle transitions and animations to enhance the user experience without overwhelming them.
